# üìù Words vs Keywords in JavaScript

JavaScript has two main types of words:

- üîë **Keywords:** Reserved words with special meaning in JavaScript. You **cannot** use them as variable, function, or property names.
- üè∑Ô∏è **Identifiers:** User-defined names for variables, functions, properties, etc. (as long as they're not keywords).

---

## ‚ùì Why Does This Matter?

> Keywords are part of the language syntax. Using them incorrectly (like as a variable name) will cause a syntax error!

---

## ‚ö° Quick Examples

```javascript
// ‚úÖ Valid identifiers
let name = 'Alice';
const PI = 3.14;
function greet() { return 'Hi'; }

// ‚ùå Invalid: 'var' is a keyword and cannot be used as an identifier
// let var = 5; // SyntaxError
```
